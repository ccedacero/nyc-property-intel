# Golden BBLs for Demos

These BBLs have rich data across multiple sources (PLUTO + HPD violations + DOB violations + DOF sales), making them ideal for demonstrating the full tool suite.

## Recommended Demo Properties

### 1. Manhattan HDFC — 170 W 85th St (Upper West Side)
- **BBL**: `1012150061`
- **Why**: 5-floor, 28-unit HDFC co-op from 1900. 77 HPD violations (7 Class C), 12 DOB violations, multiple sales. Classic Upper West Side pre-war building.

### 2. Brooklyn Walk-up — 413 9th St (Park Slope)
- **BBL**: `3010060055`
- **Why**: 6-floor, 20-unit building in Park Slope. 57 HPD violations (5 Class C), 3 DOB violations. Great for showing neighborhood-level comps in a hot market.

### 3. Bronx Multifamily — 132 W 169th St (Highbridge)
- **BBL**: `2025180028`
- **Why**: 6-floor, 25-unit building. 87 HPD violations (18 Class C), 38 DOB violations. Shows a building with significant regulatory risk — perfect for demonstrating the risk analysis.

### 4. Queens Co-op — 139-05 85 Dr (Briarwood)
- **BBL**: `4097110125`
- **Why**: 48-unit Queens co-op. 90 HPD violations (22 Class C hazardous), 25 DOB violations. High violation count makes this a great test for the issues tool.

### 5. Manhattan Harlem — 303 W 122nd St
- **BBL**: `1019490026`
- **Why**: 6-floor, 42-unit HDFC. 71 HPD violations, 38 DOB violations, multiple sales. Good for showing the full analyze_property power tool.
